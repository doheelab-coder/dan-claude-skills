# 데이트 신청글 작성 스킬

여성의 프로필 이미지를 분석하여 호감을 주는 최적의 데이트 신청글을 작성합니다.

## 경로

- **입력**: `/Users/dan.jung/Documents/code/dan-claude-skills/dan-propose/input/`
- **출력**: `/Users/dan.jung/Documents/code/dan-claude-skills/dan-propose/output/`
- **완료**: `/Users/dan.jung/Documents/code/dan-claude-skills/dan-propose/archive/`

## 나의 관심사

상대방 프로필의 취미/관심사와 겹치는 항목을 찾아 "관심사가 비슷해서 대화가 잘 통할 것 같아요" 문장에 활용합니다.

- 운동, 헬스, 수영, 건강
- 여행, 맛집, 커피
- 독서, 전시회, 공연
- 요리, 자기계발, 재테크

## 실행 방법

`/dan-propose` 명령으로 실행합니다.

```

이미지가 없으면:
```

### Step 1.5: 파일명을 닉네임으로 변경 (자동 진행)

1. Read 도구로 이미지를 읽고 닉네임 추출
2. `mv` 명령어로 파일명을 닉네임으로 변경

   - 별표(*)는 언더스코어(_)로 치환: `sum****` → `sum____.png`

### Step 2: 모든 이미지 처리 (자동 진행)
모든 PNG 파일에 대해 아래 과정을 반복합니다.

#### 2-1. 프로필 정보 추출
Read 도구로 이미지를 읽고, 다음 항목을 추출합니다:
- 닉네임, 나이, 소개, 성격, 취미, 관심사, I have, I want

#### 2-2. 데이트 신청글 작성
아래 **신청글 작성 규칙**에 따라 최적의 신청글 1개를 작성합니다.

#### 2-3. 결과 저장

- 일자별 1개 파일만 생성
- 같은 날 여러 이미지 분석 시, 기존 파일에 내용 추가

## 신청글 작성 규칙

### 작업 순서

1. **정보 추출 및 분석**
   - 이미지 내 텍스트를 사실 기반으로만 추출
   - 추출 항목: (닉네임 / 나이 / 소개 / 성격 / 취미 / 관심사 / I have / I want)
   - **소개: 이미지에 있는 소개 내용을 요약하지 말고 원본 그대로 모두 추출**
   - 추출한 내용은 맨 앞에 정리하여 보여줘

2. **신청글 작성 가이드 (4-5줄 구조)**

   ① 첫 줄 (필수 / 고정 문장): "여성스럽고 세련된 분위기가 너무 예쁘시고 제 스타일이라 용기 내어 연락드려요. 😊"

   ② 둘째 줄 (성격 칭찬): **반드시 '소개' 내용을 최우선으로 참고하여 작성** / 문장은 반드시 "더 매력적으로 느껴졌어요"로 마무리 / 문장 끝에 느낌표 붙이기

   **작성 우선순위:**
   1. 자기관리/운동 키워드가 소개에 있으면 → "자기관리를 잘 하시는 모습이" 또는 "꾸준히 자신을 가꾸시는 모습이"로 시작
      - 키워드: 자기개발, 운동, 관리, 헬스, 건강, 독서, 공부, 수영, 필라테스, 요가, 셀프케어 등
   2. 자기관리 키워드가 없으면 → 소개에 있는 구체적인 성격이나 태도 표현 사용
      - 예: 긍정적, 밝은, 진지한, 존중, 유쾌한, 솔직한 등
   3. 소개가 매우 짧거나 구체적 내용이 없을 때만 → '성격' 부분 참조

   ③ 셋째 줄 (연애관 공감 - 선택): 소개에 이성관, 연애관이 있다면 요약해서 "~점도 저랑 비슷해서 잘 맞을 것 같아요." 표현 / "또"로 문장 시작 / 없다면 생략
   - 예: "또 진지한 만남을 원하신다고 하셨는데, 그런 점이 저랑 비슷해서 잘 맞을 것 같아요."

   ④ 넷째 줄 (취미/관심사 연결): 취미와 관심사를 구체적으로 여러 개 언급하여 자연스럽게 연결 / "그리고"로 문장 시작
   - 2개 이상 나열: "그리고 저도 [취미1]이랑 [취미2], [관심사] 좋아해서 대화가 잘 통할 것 같아요."

   ⑤ 마무리 (고정 의문문): "괜찮으시면 편하게 대화하면서 천천히 알아가보는 건 어떨까요?"

### 스타일 및 제약 사항

- 전체 줄 수: 4-5줄 (첫 줄 + 성격 칭찬 + 연애관(선택) + 취미/관심사 + 마무리)
- 취미/관심사 줄은 가능한 한 길고 풍부하게 작성 (2개 이상 언급)
- 톤: 캐주얼하고 밝게. 대화 하듯 자연스럽게
- 이모지: 첫 줄에만 😊 1개 사용 (이후 사용 금지)
- 금지 사항:
  - 과장된 표현, 오글거리는 멘트, 근거 없는 추측 금지
  - 프로필에 없는 내용 칭찬 금지
  - "자연스럽다", "인상 깊다" 사용 금지
  - 줄 간 반복되는 표현 제거: 동일한 단어나 어구가 여러 줄에 걸쳐 반복되지 않도록 검토 후 다른 표현으로 대체
- 출력 전 프로필에 없는 표현이 있는지 한번 더 검토

### 출력 형식

- 최적의 신청글 1개 제시
- [이름 / 나이] 출력
- 첫 줄 제외하고 각 줄 끝날 때 마침표(.) 사용
